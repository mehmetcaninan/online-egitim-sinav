# Online EÄŸitim Frontend

React + Vite frontend for the online education platform.

## Setup

```bash
cd frontend
npm install
```

## Run Dev Server

```bash
npm run dev
```

Opens at http://localhost:5173

## Features

- **Courses**: List and create courses
- **Upload**: Upload videos/materials to courses
- **Admin**: Login and manage user roles

## API

Backend runs at http://localhost:8080
Default admin: username=admin, password=123456

## Build for Production

```bash
npm run build
```

Output in `dist/` folder.
